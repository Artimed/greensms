# Design v2 (Single Page)

> Status: `ACTIVE`
> Role: `UI/UX source of truth for MVP`
> Scope: одностраничный интерфейс и QR-flow
> Date: `2026-03-14`

## Базовые принципы
- Приложение одностраничное.
- Логин/регистрация отсутствуют (полностью локальная модель).
- Обязательный минимум настроек: телефон устройства.
- Допустимый MVP+ scope настроек: лимит, язык, тема, разрешения, очистка локальных данных и about.
- Допустимый MVP+ scope настроек: лимит, язык, тема, разрешения (`READ/RECEIVE SMS`, `CAMERA`, `SEND_SMS`), очистка локальных данных и about.
- Допустимый MVP+ scope настроек (Wave 2): выбор `страны` и `банка` для формирования платежных команд.

## Главный экран
- Экран содержит:
  - верхний блок состояния;
  - QR-баннер для перехода в QR-режим;
  - карточки счетов;
  - ленту последних SMS.
- Настройки открываются через иконку в AppBar.

## QR-режим (полноэкранный)
- Камера открывается сразу на весь экран.
- Нижняя панель действий:
  - загрузить QR из галереи;
  - кнопка «Сканировать»;
  - «QR с суммой» (телефон + сумма, `last4` опционально);
  - «По телефону» (режим без QR: ввод телефона + суммы + опционального `last4`);
  - «Мой QR» (только телефон, без суммы);
  - выйти назад.
- При формировании «QR с суммой» для Сбера:
  - сумма перевода обязательна;
  - `last4` карты/счета списания опционален;
  - блок выбора `last4` раскрывается по требованию;
  - при наличии счетов в приложении `last4` выбирается из списка карточек.
- При сканировании «Мой QR» (без суммы):
  - перед формированием SMS сумма обязательна;
  - `last4` источника изначально пустой, но доступен через раскрывающийся блок.
- После успешного сканирования:
  - сразу показывается подтверждение формирования SMS;
  - при подтверждении:
    - если включен режим прямой отправки SMS, приложение отправляет SMS на `900` напрямую;
    - если прямая отправка отключена или завершилась ошибкой, открывается системный SMS-экран;
  - при отмене пользователь остается на экране сканирования.
- Режим «По телефону»:
  - открывает такую же нижнюю шторку, как QR-ввод, но с редактируемым полем телефона;
  - после подтверждения сразу переходит к экрану подтверждения платежной команды (без сканирования QR).
